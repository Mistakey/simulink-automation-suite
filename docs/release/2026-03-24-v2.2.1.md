# Release Notes — v2.2.1

## Summary

Patch release fixing a noise issue in the `find` action output: Variant Subsystem warnings emitted by MATLAB were leaking into successful search results, causing agents to see spurious warnings on every query against models that contain variant blocks.

## Highlights

- **find: filter Variant Subsystem warnings from output** — MATLAB emits internal "Variant" warnings during `find_system` calls on models that use Variant Subsystems. These warnings are unrelated to the search query and were polluting the `warnings` field of every successful find result. They are now filtered from the success output. Non-variant warnings (e.g. link resolution warnings) continue to pass through unchanged. The full warning list is still preserved in `details.warnings` when the action returns an error, retaining debuggability.

## Compatibility / Upgrade Notes

- No breaking changes. Agents that previously filtered variant warnings from `find` output themselves will now receive clean output without extra filtering.
- `schema` version remains `"2.2"` — no contract shape changes.

## Validation

- Unit tests: 218 tests, all pass
- Live test report: archived at `docs/reports/archive/live-test-v2.2.1.md`
  - Report was generated at commit `7244e65` (pre-fix). The fix reduces output noise only; no regression risk.

## 中文说明

### 概述

此补丁版本修复了 `find` 操作输出中的噪声问题：MATLAB 在搜索含有 Variant Subsystem 的模型时会产生内部警告，这些警告之前会污染成功查询的 `warnings` 字段，导致 AI Agent 在每次搜索时都收到与查询无关的警告信息。

### 主要变更

- **find 操作：过滤 Variant Subsystem 警告** — MATLAB 对含有变体子系统的模型执行 `find_system` 时会发出内部 Variant 警告，与搜索查询本身无关。这些警告现已从成功输出中过滤。非 Variant 类警告（如链接解析警告）仍然正常透传。错误响应中的 `details.warnings` 保留完整警告列表，方便调试。

### 兼容性说明

- 无破坏性变更。之前需要自行过滤 variant 警告的 Agent 现在将直接收到干净的输出。
- Schema 版本保持 `"2.2"`，接口结构无变化。
