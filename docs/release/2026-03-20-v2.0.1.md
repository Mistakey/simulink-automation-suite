# Release v2.0.1

Release date: 2026-03-20

## Summary

`v2.0.1` is a patch release focused on transport correctness and release metadata sync.

## Highlights

- Added a unified MATLAB transport layer for runtime calls that need stable warning capture and clean JSON stdout.
- Fixed live `set_param` behavior so execute mode uses no-output writes, read-back verification, and executable rollback payloads.
- Preserved warning contracts across `scan`, `find`, `connections`, `inspect`, `highlight`, and helper-level model resolution.
- Documented JSON mode as the canonical request surface for complex strings and newline-containing values.
- Fixed rollback replay for parameters whose previous value is the empty string.

## Compatibility / Upgrade Notes

- Plugin version is now `2.0.1`.
- Schema version remains `2.0` because the plugin major.minor contract did not change.
- No migration is required for existing installs beyond updating the plugin package.
